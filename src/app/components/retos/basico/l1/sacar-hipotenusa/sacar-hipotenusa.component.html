Calcular hipotenusa

Escribir un programa que calcule la hipotenusa de un triangulo rectangulo cuyos catetos son ingresados por el usuario.

En esta solucion, lo mas optimo es importar una libreria que contiene multiples funciones matematicas, las cuales nos sirven para facilitar el trabajo evitando tener que realizar cierta cantidad de operaciones una a una, como lo son la raiz cuadrada y la potencia en este caso.
