Introduccion a GitHub

GitHub es una plataforma web que le permite a los desarrolladores publicar sus proyectos de una manera sencilla, ya que ofrece una gran cantidad de herramientas como lo son la posibilidad de trabajo colaborativo, pull requests, manejo y proteccion de ramas, entre otras funcionalidades.

Su principal caracteristica es su compatibilidad con Git, ya que permite publicar los repositorios de nuestro computador para mostrarlos a la comunidad.
