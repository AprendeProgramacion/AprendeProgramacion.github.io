Repositorio

Repositorio es el nombre tecnico de la 'base de datos' donde se guardan los cambios.

Para poder guardar los cambios, es necesario crear nuestro repositorio en el directorio (carpeta) desde donde queremos guardar. Por ejemplo, si tenemos una carpeta llamada 'PROYECTO' en el escritorio y dentro de ella iniciamos el repositorio, todos los archivos dentro de 'PROYECTO' van a ser reconocidos por git y por lo tanto almacenados.

NINGUN archivo va a ser reconocido por git si se encuentra afuera de la carpeta donde se inicializó el repositorio. Solo son reconocidos los archivos dentro del directorio y todos sus subdirectorios.

Para inicializar un repositorio, debemos movernos al directorio deseado desde la terminal y posteriormente ejecutar:

  • git init
    •

    Para verificar si todo salió bien debes ejecutar:

  • ls -a
    •

    debería salir una carpeta llamada:

  • .git/
    •

    La carpeta '.git' es una carpeta oculta que contiene toda la informacion de cambios en el repositorio, por lo tanto no se debe modificar ya que al hacerlo, podria dañar el repositorio.
