GIT

Bienvenido al curso de Git. Vas a aprender a utilizar una herramienta muy util e importante en la programacion.

En este curso vas a ver todo desde cero, no necesitas conocimientos aparte de saber que es un bloc de notas y uso basico de navegador para tomarlo.


¿Qué es GIT?

Git es un Software de control de versiones que permite a los desarrolladores tener un historial de todos los cambios que se hacen en un proyecto.

En términos sencillos es una base de datos donde se almacenan todos los cambios de nuestro proyecto de software.

Git se encarga de guardar todos los cambios, cualquier modificación que esté al alcance este software va a ser detectado y almacenado en la base de datos. Esto nos ayuda a controlar mejor nuestro código para, en caso de necesitarlo, poder volver a un punto anterior del proyecto, revisarlo, modificarlo, e incluso crear un codigo totalmente diferente.


Historia

Git fue desarrollado por Linus Torvalds, el creador del Kernel de Linux, con el fin de gestionar este último.

si bien ya se tenían herramientas para gestion de versiones, ninguna se ajustaba al proyecto de Linus, por lo que decidió crear su propio Software de control.

Así, en el año 2005 salió Git, un software innovador, rapido, multiplataforma, que ademas tuviera una manera de separar el desarrollo en diferentes lineas de tiempo.

Hoy en dia git es el sistema de cotrol de versiones mas popular, siendo usado por casi todos los desarrolladores a nivel mundial.
