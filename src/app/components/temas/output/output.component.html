Salida

Todos los procesos de un programa tienen un resultado final que se puede almacenar en la memoria del equipo o puede tener una respuesta inmediata. ¿Pero cómo acceder a esa información? Para eso existe lo que se conoce como salida de un programa, que no es nada más que lo que podemos ver. Un ejemplo es este mismo texto que estás viendo. Hay un código interno que muestra lo que contiene la pagina.

C++ es un lenguaje compilado que no permite desarrollar interfaz gráfica, por eso es que nunca vas a ver una ventana con botones hecha en C++, sino que vas a trabajar mediante una terminal. así que su salida estándar sólo será texto por consola.
