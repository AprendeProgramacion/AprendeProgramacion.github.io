Operadores Logicos

Las decisiones son un elemento muy importante en la programación, ya que podemos realizar diferentes flujos dependiendo de lo que el usuario ingrese. Por ejemplo un menú con diferentes opciones, el usuario debe seleccionar una para que el programa muestre específicamente los datos de dicha opción.

No obstante, debemos conocer un nuevo tipo de operación, estas se conocen como operaciones lógicas.

Estas operaciones devuelven un tipo de dato booleano, que como ya vimos, se resume en verdadero o falso.

Con los operadores de comparacion podemos validar expresiones, como por ejemplo si un numero es mayor que otro o si el contenido de una variable es igual a otra.

Ahora vamos a ver los operadores de conjuncion y disyuncion que nos permiten hacer validaciones entre booleanos.

Estos operadores nos permiten evaluar las expresiones de tipo booleano. De este modo, el programa puede tomar decisiones de una manera mas precisa.
