Instalacion de Git

Windows

  1. Verificamos el tipo de sistema operativo que tenemos instalado, para eso, debemos ir al explorador de archivos. Luego buscamos 'Este equipo' y le damos click derecho para seleccionar propiedades. Alli sabremos si el sistema es de 32 bits o de 64 bits.
  2. Abrimos el navegador y entramos a git-scm.com, luego damos click en el boton descargar para windows.
  3. Seleccionamos el enlace de descarga de acuerdo con nuestro tipo de sistema.
  4. Ejecutamos el instalador y le damos siguiente sin modificar ninguna configuracion.
  5. Cuando finalice la instalacion, vamos a deshabilitar la casilla 'View release notes' y habilitamos 'Launch Git Bash'.

Linux

  1. Abrimos un terminal.
  2. Ejecutamos:

    3. Debian y derivados

  3. sudo apt install git
    4.

    Arch y derivados

  4. sudo pacman -S git
    5.


Configuracion de Git

Una vez tenemos instalado git, debemos abrir la terminal correspondiente para poder hacer configuraciones basicas antes de empezar a guardar cambios.

  1. Debemos cambiar la rama por defecto que se crea cuando inicializamos un repositorio. Esto es debido a una incompatibilidad con github, ya que la rama que viene configurada es 'master', mientras que github trabaja con 'main'.
    Para ello vamos a ejecutar el siguiente comando:
  2. git config --global init.defaultBranch "main"
    3.
  3. En segundo lugar, debemos decirle a git quienes somos, esto les ayuda a los otros miembros del equipo a saber quien hizo los cambios.
    Para esto ejecutamos:
  4. git config --global user.name "{{'<'}}nombre{{'>'}}"
    5.
  5. git config --global user.email "{{'<'}}email{{'>'}}"
    6.
