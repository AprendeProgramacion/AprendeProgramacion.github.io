Operadores Aritmeticos

Con la programación también podemos realizar operaciones matemáticas básicas, como la suma, resta, multiplicación y división.

A partir de estas cuatro operaciones junto con el modulo (lo vamos a explicar bien un poco mas adelante), podemos crear otros algoritmos mas complejos como lo son por ejemplo potencias.

A continuación te explico como puedes usar los operadores aritméticos y el módulo en un programa.

También existe un orden para resolver las operaciones:

  1. ( )
  2. * / %
  3. + -

No solo podemos hacer operaciones aritméticas, sino que también podemos asignar valores a las variables con base en estos operadores. A estos nuevos tipos de operadores se les conoce como operadores de asignación.

Los operadores de asignación nos ayudan a cambiar el valor de las variables de manera sencilla, ya que acortan la manera como se escriben las operaciones.
